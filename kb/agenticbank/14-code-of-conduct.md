# Code of Conduct & Ethics — PSL-014

**Version:** 5.0  **Effective:** 2026-01-01  **Owner:** HR / Legal

## 1. Pillars
- Integrity in every dealing
- Customer fairness
- Confidentiality
- Conflict of interest disclosure
- Speak-up culture

## 2. Gifts & Entertainment
| Value | Action |
|---|---|
| Below EUR 50 | Permitted, no declaration |
| EUR 50 – 250 | Declare in register |
| Above EUR 250 | Pre-approval required |
| Cash or cash-equivalent | Never accepted |

## 3. Personal Account Dealing
Pre-clearance required for all securities trades; 30-day holding period.

## 4. AI Tool Use by Employees
Employees may not paste customer-identifiable data into unsanctioned AI tools.
Sanctioned tools (e.g. ForgedScholarTwo, ForgedAgentOne) apply Purview
classification automatically.
